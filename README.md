# Random Color Changer

A simple interactive web app that changes the background color of the page to a random color on every button click.

## Live Demo

[Click here](https://priyanshuroydev.github.io/Simple-Color-Changer-Website/)
---

## How It Works

- Click the **"Change Color"** button
- The background instantly changes to a randomly generated color
- The hex code of the current color is displayed on screen
- Every click gives a completely new color
